# 냉잔고



# 프로젝트 소개
- 냉장고의 재료를 촬영하여 냉장고의 재료를 활용하여 레시피를 추천합니다.
- 집에 남아있는 음식 등 재료를 사진으로 찍어 관련 음식을 찾을 수 있습니다.
- 찾은 음식의 레시피를 확인할 수 있습니다.
- 게시판을 활용하여 질문을 하거나 내가 만든 음식을 자랑할 수 있습니다.
